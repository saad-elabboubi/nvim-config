# Neovim Fast Path

Leader is `Space`.

## Open The Main Workflows

- `Space ff` find files
- `Space fg` search text in the project
- `Space fb` switch buffers
- `Space fr` recent files
- `Space e` directory editor
- `-` open parent directory
- `Space gg` Git UI with LazyGit
- `Space tt` terminal
- `Space xx` diagnostics panel
- `Space fk` search keymaps

## Code

- `gd` go to definition
- `gr` find references
- `gi` go to implementation
- `K` hover docs
- `Space ca` code action
- `Space rn` rename symbol
- `Space cf` format file
- `[d` / `]d` previous / next diagnostic
- `Space cd` show diagnostic on current line

## Splits And Buffers

- `Space sv` vertical split
- `Space sh` horizontal split
- `Space sx` close split
- `Ctrl-h/j/k/l` move between Neovim splits and tmux panes
- `Space bn` next buffer
- `Space bp` previous buffer
- `Space bd` delete buffer

## Tmux

- `Ctrl-Space v` split right
- `Ctrl-Space b` split below
- `Ctrl-Space f` fullscreen current pane
- `Ctrl-Space q` close current pane
- `Ctrl-Space e` new window/tab
- `Ctrl-Space a` previous window/tab
- `Ctrl-Space d` detach session
- `Ctrl-Space r` reload tmux config
- `Ctrl-a` prefix fallback

## Mistral Chat Project

- `Space pb` backend checks: ruff, format, basedpyright, smoke import
- `Space pf` frontend checks: lint and TypeScript
- `Space pa` all checks
- `Space pd` frontend dev server

## Terminal Mode

- `Esc Esc` leave terminal mode
